# 降雨頻率分析
- 提供常態分布、對數常態分布、皮爾森第三類分布、對數皮爾森第三類分布、極端值第一類分布
- 卡方統計檢定、KS統計檢定
- 可上傳檔案或貼上csv格式的資料
- 使用時請註明出處
- 請貼上 CSV 格式的雨量資料。第一行為標題，第一欄為年份。範例如下：
```json

```
